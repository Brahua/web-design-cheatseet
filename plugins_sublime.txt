Plugins de SublimeText:

- Alignment: Seleccionar todas las lineas a alinear y presionar Ctrl+Alt+A
- ColorPicker: Ctrl+Shift+C
- DocPHPManualer: Las teclas de acceso rápido predeterminadas son ctrl + alt + d para Mostrar definición y ctrl + alt + s para Buscar manual
- HTMLBeautify: Abrir un archivo HTML, luego ir a la pestaña EDITAR luego seleccionar Beautify HTML.
